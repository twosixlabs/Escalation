Advanced Plotly

You can also add anything allowed by plotly into the config wizard. All of the available option are on the plotly website. For example we want to make the histogram have an opacity of 0.5. According to the plotly figure reference opacity is a key in the dictionary of the data list.

  • To add, we would click the properties button next to data dictionary 1.
  • Type in "opacity" into the input box, click add, and click the properties button again to close it.
  • Then we have a new key called opacity, we set it to number and type in 0.5.
